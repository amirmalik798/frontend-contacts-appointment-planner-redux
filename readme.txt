
APPOINMENT PLANNER
Developed by: Amir Malik
Date: 01-05-2026

This application has two main features:
1) Contacts
2) Appointments

Contacts Feature:
-> It allows the user to create and store contacts.
-> Contacts are the stored in Local Storage.
-> It resists duplicate creation of contacts with same phone# and/or email.
-> For successful contact creation, both phone# and email should not be present before.
-> Contacts once created, can be edited and deleted.

Appointments Feature:
-> It allows the user to create and store appointments.
-> Appointments are stored in Local Storage.
-> It resists duplicate creation of appointments with same title, contact, date & time.
-> It resists creation of appointments on same day at same time.
-> Appointments once created, can only be deleted.
-> Creation of an appointment requires a Contact.


Created using: React & Redux Toolkit
